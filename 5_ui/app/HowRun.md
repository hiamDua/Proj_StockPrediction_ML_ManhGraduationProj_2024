# Cài đặt môi trường ảo

1. Tạo môi trường ảo:
   ```bash
   python -m venv .venv
   ```

2. Di chuyển vào thư mục ứng dụng:
   ```bash
   cd 5_ui/app
   ```

3. Cài đặt các thư viện cần thiết:
   ```bash
   pip install -r requirements.txt
   ```

4. Chạy ứng dụng Streamlit:
   ```bash
   streamlit run app.py
   ```
